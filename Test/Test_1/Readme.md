# Solar Environmental System

## Goal
To build a real-world sensing system that collects environmental data and studies its effect on solar energy efficiency and system optimization.

## Current Stage
Testing sensors and collecting raw data.

## Hardware
- ESP32-S3
- Light sensor (Lux)
- Temperature sensor
- Barometric sensor
- Solar panel + battery

## Current Problems
- Sensor noise
- Invalid readings
- Wiring instability
- Environmental interference (dust, insects)
